# Branch Performance Dashboard

A React-based dashboard that displays branch performance data from Google Sheets.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- Real-time data fetching from Google Sheets
- Responsive KPI cards
- Interactive branch performance table
- Color-coded performance indicators
- Auto-updating dashboard

## Data Source

The dashboard fetches data from your published Google Sheets CSV.
